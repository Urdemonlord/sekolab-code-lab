# Kurikulum HTML & CSS - Panduan Pembelajaran Komprehensif

```markdown
# Context untuk GitHub Copilot VSCode

## INSTRUKSI PEMAHAMAN KONTEKS

Ini adalah kurikulum pembelajaran HTML dan CSS komprehensif yang mencakup dari tingkat dasar hingga lanjutan. Setiap modul dirancang secara sistematis dengan fokus pada pembelajaran praktis dan implementasi nyata.

## PARAMETER PEMBELAJARAN

- Bahasa Instruksi: Bahasa Indonesia
- Target Pembelajar: Pemula hingga Tingkat Menengah
- Durasi per Modul: 1-2 minggu
- Metode: Pembelajaran Berbasis Proyek
- Output: Kode yang Bersih, Terstruktur, dan Mengikuti Best Practices

## ATURAN PENULISAN KODE

1. Gunakan indentasi 2 spasi
2. Ikuti standar HTML5
3. Terapkan BEM naming convention untuk CSS
4. Prioritaskan pendekatan mobile-first
5. Optimalkan untuk performa dan aksesibilitas
6. Sertakan komentar yang informatif
7. Validasi HTML dan CSS

## KONVENSI PENAMAAN

- HTML Classes: kebab-case
- CSS Variables: --nama-variabel
- File Names: kebab-case
- Component Files: ComponentName.css

## PANDUAN RESPONSIF

- Mobile: 320px - 480px
- Tablet: 481px - 768px
- Desktop: 769px - 1024px
- Large: 1025px+

## CHECKLIST KUALITAS KODE

- [ ] Validasi W3C
- [ ] Cross-browser compatibility
- [ ] Responsive design
- [ ] Optimized assets
- [ ] Semantic HTML
- [ ] Clean CSS
- [ ] Documented code
- [ ] Accessibility compliance

## BEST PRACTICES

### HTML
- Gunakan tag semantik
- Optimalkan struktur heading
- Sertakan alt text untuk gambar
- Validasi form client-side
- Gunakan label untuk form elements

### CSS
- Hindari !important
- Gunakan CSS custom properties
- Terapkan progressive enhancement
- Optimalkan selector specificity
- Kelompokkan related styles

## DEBUGGING GUIDELINES

1. Inspect element untuk troubleshooting
2. Validasi HTML/CSS
3. Check browser compatibility
4. Test responsive breakpoints
5. Optimize performance
```
