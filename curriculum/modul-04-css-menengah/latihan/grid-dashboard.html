üìù Instruksi Latihan Grid Layout

Lengkapi CSS untuk membuat layout dashboard menggunakan CSS Grid. Ikuti langkah-langkah berikut:

  1. Dashboard Container (.dashboard)
    • Gunakan display: grid
    • Definisikan grid-template-areas: 
      "header header"
"sidebar main"
"footer footer"
    • Atur grid-template-columns: 250px 1fr
    • Atur grid-template-rows: auto 1fr auto
    • Tambahkan min-height: 100vh
  2. Assign Grid Areas
    • .header ‚Üí grid-area: header
    • .sidebar ‚Üí grid-area: sidebar
    • .main ‚Üí grid-area: main
    • .footer ‚Üí grid-area: footer
  3. Content Grid (.content-grid)
    • Buat nested grid untuk content cards
    • Gunakan grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))
    • Tambahkan gap untuk spacing
  4. Responsive Design
    • Pada mobile (max-width: 768px), ubah grid menjadi: 
      "header"
"main"  
"sidebar"
"footer"
    • Gunakan grid-template-columns: 1fr
üí° Hint: CSS Grid sangat powerful untuk layout 2D. Gunakan grid-template-areas untuk membuat layout yang mudah dibaca dan diubah!
