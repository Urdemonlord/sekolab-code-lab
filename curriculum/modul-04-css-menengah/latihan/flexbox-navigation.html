Latihan Flexbox Navigation

Instruksi Latihan

Lengkapi CSS untuk membuat navigation bar yang responsive menggunakan Flexbox. Ikuti langkah-langkah berikut:

  1. Header (.header)
    • Berikan background color putih
    • Tambahkan box-shadow untuk depth
    • Gunakan position: sticky agar tetap di atas saat scroll
  2. Navbar (.navbar)
    • Gunakan display: flex
    • Atur justify-content: space-between untuk spacing
    • Atur align-items: center untuk vertical alignment
    • Tambahkan padding yang sesuai
  3. Brand (.navbar__brand)
    • Style sebagai logo/brand dengan warna menarik
    • Hilangkan text-decoration
    • Berikan font-weight bold dan font-size yang lebih besar
  4. Navigation Menu (.navbar__nav)
    • Gunakan display: flex
    • Hilangkan list-style
    • Tambahkan gap antar menu items
  5. Navigation Links (.navbar__link)
    • Hilangkan text-decoration
    • Berikan warna yang sesuai
    • Tambahkan hover effects dengan transition
  6. Action Buttons (.navbar__actions)
    • Gunakan flex untuk layout
    • Tambahkan gap antar buttons
  7. Button Styles (.btn)
    • Style dasar: padding, border-radius, cursor
    • Buat variant .btn--primary dan .btn--outline
    • Tambahkan hover effects
  8. Responsive Design (Media Query)
    • Sembunyikan menu navigation pada mobile (max-width: 768px)
    • Sesuaikan layout untuk layar kecil

Hasil yang Diharapkan

  • Navigation bar yang rapi dengan logo di kiri, menu di tengah, dan buttons di kanan
  • Hover effects pada links dan buttons
  • Responsive design yang menyembunyikan menu pada mobile
  • Sticky navigation yang tetap terlihat saat scroll
