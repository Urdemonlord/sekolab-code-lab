📝 Template Latihan HTML Dasar

📋 Instruksi:

File ini adalah template untuk latihan HTML Anda. Ikuti petunjuk di bawah ini dan ganti bagian yang bertanda [TULIS KODE DI SINI] dengan kode HTML yang sesuai.

🎯 Latihan 1: Heading dan Paragraf

Tugas: Buat struktur heading (h1-h3) dan beberapa paragraf tentang diri Anda

🎯 Latihan 2: Pemformatan Teks

Tugas: Gunakan tag <strong>, <em>, <mark>, dll untuk memformat teks

🎯 Latihan 3: Membuat List

Tugas: Buat unordered list (hobi) dan ordered list (langkah belajar coding)

🎯 Latihan 4: Link dan Gambar

Tugas: Tambahkan link ke website favorit dan gambar (gunakan placeholder)

🎯 Latihan 5: Tabel Sederhana

Tugas: Buat tabel yang berisi informasi tentang keterampilan Anda

🎯 Latihan 6: Form Kontak Sederhana

Tugas: Buat form dengan input nama, email, dan textarea untuk pesan

🎯 Latihan Bonus: Kombinasi Semua Elemen

Tugas: Buat bagian "Tentang Saya" yang mengkombinasikan semua elemen yang sudah dipelajari

✅ Checklist Setelah Selesai:

  • Pastikan semua latihan sudah dikerjakan
  • Buka file di browser untuk melihat hasilnya
  • Validasi HTML menggunakan W3C Validator
  • Perbaiki error jika ada
